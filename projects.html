Cadastro de produtos

  • Php, Laravel, CSS

Desenvolvi um sistema de cadastro de produtos com foco em objetividade e uma interface simples e intuitiva. O aplicativo permite o registro de informações essenciais, como nome do produto, descrição, categoria, valor do produto, quantidade em estoque e imagens do produto. O sistema possibilita o cadastro de qualquer tipo de produto, com a opção de anexar múltiplas imagens, que são armazenadas de forma segura no banco de dados MySQL.

A cada imagem adicionada, o sistema realiza o registro automático, garantindo que todas as informações estejam devidamente organizadas e acessíveis. Essa abordagem proporciona uma experiência eficiente para o usuário, aliando simplicidade no uso com robustez no armazenamento dos dados.

  • Desenvolvi um sistema em C para compra e gestão de ingressos com regras específicas: crianças e idosos são isentos, enquanto menores de 16 anos pagam meia entrada.
